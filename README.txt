INTRODUCTION

hum_bedework displays calendar information from IT Services Bedework calendar.


INSTALLATION

hum_bedework requires the iCalcreator library and the Libraries API (version 2.0).

The iCalcreator library can be downloaded from http://kigkonsult.se/downloads/index.php#iCalcreator. After extracting the files copy iCalcreator.class.php to a directory named sites/all/libraries/iCalcreator. Then clear the caches on your site.
